##	开始使用

1. 添加CSS样式
  使用本框架前，请在CSS中开启border-box

  ```
  *::before,*::after,*{box-sizing: border-box;}
  ```
  IE8 及以上浏览器都支持此样式。

  你还需要设置默认颜色等变量（后续会改为 SCSS 变量）
  ```
  html {
    --button-height: 32px;
    --font-size: 14px;
    --button-bg: white;
    --button-active-bg: #eeeeee;
    --border-radius: 4px;
    --color: #333333;
    --border-color: #999999;
    --border-color-hover: #666;
  }
  ```
  IE15 及以上浏览器都支持此样式。
